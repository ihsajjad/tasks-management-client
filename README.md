# Task Management

This is a small project. I made this project for the Internshala job task. This is a MERN-stack project. Users can Post, Read, Update, and Delete tasks.

### Technologies are used:

- MERN
- React-Bootstrap
- Formik
- Yup (for form validation)
- Sweetalert2
  -React Icons
  -Axios
  -Tanstack Query
